# MFG Platform 业务流程图（带事件链路）

## 一、核心事件定义

| 事件名 | 源模块 | 说明 |
|--------|--------|------|
| MATERIAL_CREATED | PLM | 物料创建 |
| BOM_REVISED | PLM | BOM修订 |
| BOM_ACTIVATED | PLM | BOM激活 |
| ROUTING_REVISED | PLM | 工艺路线修订 |
| ECN_EXECUTED | PLM→MES | ECN执行完成 |
| PURCHASE_ORDER_APPROVED | ERP | 采购订单审批通过 |
| RECEIPT_CONFIRMED | SCM | 采购收货确认 |
| RECEIPT_PO_UPDATE_REQUEST | SCM | SCM内部：收货后更新PO收货数量 |
| SALES_ORDER_CONFIRMED | ERP | 销售订单确认 |
| WORK_ORDER_RELEASED | APS→MES | 工单下达 |
| PRODUCTION_STARTED | MES | 生产开始 |
| PRODUCTION_COMPLETED | MES | 生产完工 |
| MATERIAL_ISSUE_REQUEST | MES | 领料请求 |
| MATERIAL_ISSUED | WMS | 物料出库完成 |
| STOCK_IN_COMPLETED | WMS | 库存入库完成 |
| INSPECTION_TRIGGERED | MES | 质检触发 |
| INSPECTION_COMPLETED | QMS | 质检完成 |
| EQUIPMENT_STATUS_CHANGED | EAM | 设备状态变更 |
| EQUIPMENT_FAILURE_REPORTED | MES | 设备故障上报 |
| OPERATION_COMPLETED | MES | 工序完成 |
| WORK_ORDER_ALL_COMPLETED | MES | 工单全部完工 |
| FQC_INSPECTION_REQUEST | MES | FQC质检请求 |
| PRODUCTION_RECEIPT_REQUEST | MES | 生产入库请求 |
| WO_RECEIPT_WRITEBACK | MES | 工单入库回写 |
| CHILD_WO_COMPLETED | MES | 子工单完工 |
| WO_CANCELLED | MES | 工单取消 |
| OUTSOURCING_ISSUE_CONFIRMED | OUTSOURCING | 外协发料确认 |
| OUTSOURCING_RECEIPT_CONFIRMED | OUTSOURCING | 外协收货确认 |
| OUTSOURCING_RECEIPT_STOCKED | WMS | 外协检验通过正式入库 |
| OUTSOURCING_SETTLEMENT_APPROVED | OUTSOURCING | 外协结算审核通过 |
| OUTSOURCING_PAYABLE_CREATED | SCM | 外协应付账款创建 |
| CERTIFICATION_UPDATED | HR | 技能认证变更 |
| OPERATION_REPORTED | MES→HR | 报工完成 |
| RECALL_FREEZE_REQUEST | TRACEABILITY | 召回冻结请求 |
| RECALL_FREEZE_COMPLETED | WMS | 召回冻结完成 |

## 二、事件订阅关系

| 事件名 | 订阅模块 | 处理逻辑 |
|--------|---------|---------|
| WORK_ORDER_RELEASED | MES | 接收APS工单下达事件，创建MES工单并建立父子关联 |
| OPERATION_COMPLETED | MES | 触发自动入库编排 |
| WORK_ORDER_ALL_COMPLETED | MES | 触发自动入库编排；若存在父工单则发布CHILD_WO_COMPLETED |
| STOCK_IN_COMPLETED | MES | 触发自动入库编排 |
| INSPECTION_COMPLETED | MES | 触发自动入库编排 |
| CHILD_WO_COMPLETED | MES | 更新物料齐料状态(MaterialReadinessService) |
| WO_CANCELLED | MES | 更新物料齐料状态(MaterialReadinessService) |
| ECN_EXECUTED | MES | 更新工单BOM/Routing版本或发送待审核通知 |
| RECEIPT_CONFIRMED | QMS | 自动创建IQC检验任务 |
| FQC_INSPECTION_REQUEST | QMS | 自动创建FQC检验任务 |
| OUTSOURCING_RECEIPT_CONFIRMED | QMS | 创建外协IQC检验单 |
| RECEIPT_CONFIRMED | TRACEABILITY | 创建采购原材料TraceBatch批次记录 |
| PRODUCTION_COMPLETED | TRACEABILITY | 创建成品TraceBatch+TraceLink追溯链 |
| MATERIAL_ISSUED | TRACEABILITY | 确认/创建原材料TraceBatch，标记为已消耗 |
| STOCK_IN_COMPLETED | TRACEABILITY | 更新TraceBatch.inventoryStatus=IN_STOCK |
| INSPECTION_COMPLETED | TRACEABILITY | 更新TraceBatch.inspectionStatus |
| SALES_ORDER_CONFIRMED | TRACEABILITY | 预关联销售订单到TraceBatch |
| RECALL_FREEZE_COMPLETED | TRACEABILITY | 更新TraceBatch.isFrozen=1，标记冻结 |
| MATERIAL_ISSUE_REQUEST | WMS | 执行自动出库 |
| PRODUCTION_COMPLETED | WMS | 执行自动生产入库 |
| PRODUCTION_RECEIPT_REQUEST | WMS | 执行生产入库(自动入库编排) |
| INSPECTION_COMPLETED | WMS | FQC通过则执行生产入库；IQC通过则解冻库存；外协IQC通过则暂存转正式库存 |
| OUTSOURCING_ISSUE_CONFIRMED | WMS | 执行外协出库 |
| OUTSOURCING_RECEIPT_CONFIRMED | WMS | 执行外协暂存入库 |
| RECALL_FREEZE_REQUEST | WMS | 冻结在库批次，发布RECALL_FREEZE_COMPLETED |
| RECEIPT_PO_UPDATE_REQUEST | SCM | 更新PO收货数量 |
| OUTSOURCING_SETTLEMENT_APPROVED | SCM | 创建应付账款，发布OUTSOURCING_PAYABLE_CREATED |
| OPERATION_REPORTED | HR | 汇总工时(WorkHourService) |
| MATERIAL_ISSUED | OUTSOURCING | 回填wms_tx_id到发料单 |
| OUTSOURCING_RECEIPT_STOCKED | OUTSOURCING | 更新收货单状态为PASSED，累加已检验合格数量 |
| INSPECTION_COMPLETED | OUTSOURCING | 检验不合格则更新收货单状态为FAILED |
| OUTSOURCING_PAYABLE_CREATED | OUTSOURCING | 回填scm_payable_id到结算单 |
| SALES_ORDER_CONFIRMED | APS | 触发APS排程 |
| EQUIPMENT_STATUS_CHANGED | APS | 触发APS排程重算 |
| EQUIPMENT_STATUS_CHANGED | APS(EAM) | 触发APS排程重算 |

## 四、缺失事件记录

| 缺失事件 | 触发场景 | 状态 |
|---------|---------|------|
| `EQUIPMENT_REPAIRED` | EAM维修完成 → MES恢复生产 | ⚠️ 待补充 |
| `WORK_ORDER_CHANGED` | MES工单优先级/交期变更 | ⚠️ 待补充 |
| `APS_SCHEDULE_UPDATED` | APS排程调整反馈MES | ⚠️ 待确认 |
| `SALES_ORDER_SHIPPED` | ERP销售发货完成 | ⚠️ 事件常量缺失 |
| `PURCHASE_INVOICE_CREATED` | ERP采购发票生成 | ⚠️ 待补充 |
| `PAYMENT_COMPLETED` | ERP付款完成 | ⚠️ 待补充 |

## 五、关键业务流测试检查清单

### 5.1 生产执行主链路
- [ ] ERP 销售订单确认 → SALES_ORDER_CONFIRMED
- [ ] APS 接收事件 → MRP 运算
- [ ] APS 排程 → WORK_ORDER_RELEASED
- [ ] MES 接收工单 → 创建 MES 工单
- [ ] MES 开工报工 → OPERATION_REPORTED → HR 工时汇总
- [ ] MES 工序完工 → OPERATION_COMPLETED → 自动入库编排
- [ ] FQC 质检 → INSPECTION_COMPLETED → 入库
- [ ] WMS 入库 → STOCK_IN_COMPLETED → 追溯批次创建

### 5.2 采购到生产链路
- [ ] ERP 采购订单审批 → PURCHASE_ORDER_APPROVED
- [ ] SCM 收货确认 → RECEIPT_CONFIRMED
- [ ] QMS 创建 IQC 检验
- [ ] 检验完成 → 解冻库存
- [ ] WMS 入库 → 追溯批次创建

### 5.3 父子工单联动
- [ ] 子工单完工 → CHILD_WO_COMPLETED
- [ ] 父工单齐套状态更新
- [ ] 全部完工 → 工单 COMPLETED

### 5.4 外协加工
- [ ] 发料确认 → OUTSOURCING_ISSUE_CONFIRMED
- [ ] WMS 出库 → MATERIAL_ISSUED
- [ ] 收货确认 → OUTSOURCING_RECEIPT_CONFIRMED
- [ ] WMS 暂存入库
- [ ] QMS 检验 → INSPECTION_COMPLETED
- [ ] WMS 正式入库 → OUTSOURCING_RECEIPT_STOCKED
- [ ] 结算审核 → OUTSOURCING_SETTLEMENT_APPROVED
- [ ] SCM 创建应付 → OUTSOURCING_PAYABLE_CREATED

### 5.5 设备故障
- [ ] 故障上报 → EQUIPMENT_FAILURE_REPORTED
- [ ] EAM 状态变更 → EQUIPMENT_STATUS_CHANGED
- [ ] APS 重算排程
- [ ] ⚠️ 维修完成 → MES 恢复生产 (缺失)

### 5.6 召回
- [ ] 召回发起 → RECALL_FREEZE_REQUEST
- [ ] WMS 冻结批次 → RECALL_FREEZE_COMPLETED
- [ ] TRACEABILITY 更新冻结状态
